## ✨ Features

- 🎨 **Premium Card Generator** - Generate beautiful admin cards
- 👤 **Profile Picture** - Auto-fetch Facebook profile pictures
- 🟢 **Admin Promotion** - Green/Gold theme for promotions
- 🔴 **Admin Demotion** - Red/Dark theme for demotions
- 🖼️ **Custom Backgrounds** - Different backgrounds for promote/demote
- ⚡ **Fast & Lightweight** - Built with Express.js
- 🔒 **Secure** - Environment variables for sensitive data
- 📱 **GoatBot Ready** - Designed for GoatBot V2 integration
